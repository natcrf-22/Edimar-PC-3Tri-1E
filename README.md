# Poema
   **Não Há Vagas**

 _O preço do feijão não cabe no poema_

 _O preço do arroz não cabe no poema_

_Não cabem no poema o gás, a luz, o telefone_

_A sonegação do leite_

_Da carne_

_Do açúcar_

_Do pão_

_O funcionário público_

_Não cabe no poema_

_Com seu salário de fome_

_Sua vida fechada_

_Em arquivos._
